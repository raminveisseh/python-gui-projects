# Python GUI Projects

A collection of desktop applications built with Python and PyQt6, developed during coursework in Information Technology at NC State University. Projects demonstrate progression from basic UI design to database-connected, multi-window applications.

---

## Projects

### [finalProject-Fair_Ticket_Sales-System](./finalProject-Fair_Ticket_Sales-System) — Fair Ticket Sales System
The most comprehensive project in this portfolio. A full multi-window desktop application connected to a MySQL database. Features a central dashboard that launches separate windows for purchasing tickets, searching transactions, and displaying all sales records. Demonstrates multi-window architecture, SQL INSERT/SELECT/AVG queries, tax calculations, file-based price loading, and robust error handling throughout.

**Skills:** PyQt6, MySQL, pymysql, multi-window navigation, SQL queries, file I/O, error handling

---

### [projectFour](./projectFour) — Student Grade Management System
A tabbed application that inserts and retrieves student records from a MySQL database. Automatically converts numeric scores to letter grades and displays them in a formatted list. Includes a data export feature that writes results to a text file.

**Skills:** PyQt6, MySQL, pymysql, tab navigation, data export, grade logic

---

### [financialPlanner](./financialPlanner) — Financial Planner (Time to Goal)
Calculates how many years it will take to reach a target portfolio amount based on annual deposit and a selected retirement plan (Plan 2040/2050/2060), each with different interest rates.

**Skills:** PyQt6, compound interest logic, combo box selection, while loops

---

### [financialPlanner2](./financialPlanner2) — Financial Planner (Year-by-Year Growth)
Displays a year-by-year breakdown of portfolio growth based on annual deposit amount and risk level (5% standard, 7% high risk). Outputs a running balance for each year in a list widget.

**Skills:** PyQt6, compound interest, checkbox input, formatted list output

---

### [mortalCombat](./mortalCombat) — Mortal Combat Player Manager
The only object-oriented project in this collection. Defines a `Player` class with a constructor and a `modify_stats()` method. Users can create fighters with custom speed, power, and agility attributes, then select and modify existing players from a list.

**Skills:** PyQt6, OOP, classes, constructors, instance methods, dictionary-based object storage

---

### [projectThree](./projectThree) — Home Purchase Application
A two-tab home buying application that loads property prices from an external file and calculates a full 120-month mortgage amortization schedule. Users select a house type (Apartment, Townhouse, Single-Family Home, Luxury Villa), choose a model upgrade (Standard, Smart +15%, Luxury +20%), and enter a down payment. The app computes the monthly payment using a real loan amortization formula and displays the remaining balance after each payment. Purchases are saved to a file and displayed in the second tab.

**Skills:** PyQt6, loan amortization formula, file I/O, tab navigation, radio buttons, dictionary lookups, error handling

---

### [projectTwo](./projectTwo) — Gym Membership Calculator
Calculates first-month and ongoing monthly fees based on membership plan (Basic, Standard, Premium, Family) and student discount eligibility. Includes full input validation and a clear form function.

**Skills:** PyQt6, conditional logic, discount calculations, form validation

---

### [ticketPurchase](./ticketPurchase) — Ticket Purchase System
A two-tab application where users purchase concert tickets by selecting type and quantity. Prices are loaded from an external text file into a dictionary at startup. Purchases are saved to a file and displayed in the second tab when selected.

**Skills:** PyQt6, file I/O, dictionary data structures, tab navigation, radio buttons

---

### [projectOne](./projectOne) — Ramin's Coffee Shack
A coffee order calculator for lattes, cappuccinos, and espressos. Handles size upcharges (Large adds $2 per drink) and optional tip. One of the first projects built in the course.

**Skills:** PyQt6, arithmetic logic, optional input handling

---

### [gradeCalculator](./gradeCalculator) — Grade Calculator
Converts a numeric student score (0–100) to a letter grade. Includes input validation to reject non-numeric entries and out-of-range values, displaying appropriate error messages via dialog boxes.

**Skills:** PyQt6, input validation, conditional logic, QMessageBox

---

### [fruitPurchase](./fruitPurchase) — Fruit Purchase Calculator
Calculates the cost of purchasing fruit by the pound. Dynamically displays the price per pound when a fruit is selected from a dropdown. Prices are stored in a dictionary.

**Skills:** PyQt6, dictionary lookups, combo box events, dynamic label updates

---

### [multiFormExample](./multiFormExample) — Multi-Window Communication Demo
Demonstrates how to pass data between two separate application windows in PyQt6. A user enters text in the first window and transfers it to a field in the second window.

**Skills:** PyQt6, multi-window communication, object references

---

## Technologies Used
